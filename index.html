Welcome to DownendMC




What is DownendMC?



DownendMC thrives to be an active, fun, and friendly server, made for my community and friends. With a hand-made spawnpoint and constant activity, Downendmc is a good quality survival server for everyone.. If you want to join, read the answers to questions below.



When is downendMC online?

DownendMC comes online every time a whitelisted player attempts to join. These whitelisted players are my few closest friends or server members. If you want a specific time for it to be online, feel free to request that from the owner of an Operator.



How do I join DownendMC?

For Minecraft: Java Edition


Open Minecraft Java Edition, go into the multiplayer tab, press, "add server". There should be two places you can type.
For Server Name, put, "DownendMC".
For the IP/bottom input, put exactly this: "survival.downendmc.com"
When you want to play, you can simply press play like you would for any server.



For Minecraft: Bedrock Edition (More difficult)


Download "Bedrock Together" on your phone (~1M downloads), and open it.
In the IP: section type, "survival.downendmc.com"
In the Port: section type, "60672"
After this, press Add to favourite hen "Watch Ad and Run".
Open Minecraft and navigate to the Friends tab. Connect to the server using the LAN tab. Close BedrockTogether app after the client has joined the server. (Official Bedrock Together instructions)


Minecraft is introducing a new UI, where LAN servers are now located in the Worlds tab instead of the Friends tab. To view LAN servers, ensure that LAN server visibilty is turned ON.
